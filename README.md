# User Authentication & Real-time Database System

An interactive web application featuring a modern sliding UI for **Sign In** and **Sign Up**, seamlessly integrated with a **MySQL** database for real-time user management.

---

## Project Overview
This project provides a clean, responsive, and secure user authentication flow. Users can register new accounts or log in using their **Full Name** and **Password**, with live database updates rendered instantly without full page reloads.

---

## Key Features
* **Smooth UI Animations:** Sliding transition between Sign In and Sign Up forms.
* **User Registration:** Collects Name, Age, Email, and Password.
* **Authentication:** Fast login verification via Name & Password.
* **Live Data Display (SELECT View):** Fetching and displaying registered users in a clean, formatted table.
* **Live Database:** Instant table updates using AJAX & PHP backend.
* **Status Control:** Toggle user status (Active/Inactive) dynamically.
